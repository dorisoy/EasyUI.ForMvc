This example shows the client-side events supported by EasyUI Editor for ASP.NET MVC.

The Editor has the following events:

  • OnLoad - raised when the component is initialized
  • OnChange - raised when the text is changed and editable area lost focus
  • OnExecute - raised when a command is executed
  • OnPaste - raised when the user pastes content
  • OnSelectionChange - raised when text is selected or editable area is focused

You can subscribe to each of the events in the following ways:

  • With a pre-defined javascript function 
    <%= Html.EasyUI().Editor()
        .Name("Editor")
        .ClientEvents(events => events
            .OnLoad("onLoad")
        )
%>

<script type="text/javascript">
    function onLoad(e) {
        // your code
    }
</script>
  • With an inline function, serialized from the server 
    <% Html.EasyUI().Editor()
       .Name("Editor")
       .ClientEvents(events => events
       .OnLoad(() => 
       {%>
            function(e) {
                // your code
            }
       <%})
       .Render();
%>
