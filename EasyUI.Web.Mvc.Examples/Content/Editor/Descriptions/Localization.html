This example shows the localize EasyUI Editor for ASP.NET MVC.

You can use any of the following ways to localize the editor:

  • Set the UICulture property of the Page directive: 
    <%@ Page Language="C#" Inherits="System.Web.Mvc.ViewPage" UICulture="de-DE" %>
  • Set the CurrentUICulture property of the CurrentThread in the action method: 
    public ActionResult Index()
{
    System.Threading.Thread.CurrentThread.CurrentUICulture = new System.Globalization.CultureInfo("de-DE");
}
  • Use the Localizable method of the editor: 
    <%= Html.EasyUI().Editor()
        .Name("Editor")
        .Localizable("de-DE")
%>
