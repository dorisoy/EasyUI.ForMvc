This example shows the client-side events supported by EasyUI Chart for ASP.NET MVC.

You can attach an event handler using either of these options:

  1. Specifiy the JavaScript event handler name: 
    Html.EasyUI().Chart(Model)
    .Name("chart")
    .Series(series => {
        series.Bar(s => s.RepSales).Name("Representative Sales");
        series.Bar(s => s.TotalSales).Name("Total Sales");
    })
    .ClientEvents(events => events
        .OnLoad("onLoad")
    )
  2. Declare the event handler inline (WebForms syntax): 
    <%= Html.EasyUI().Chart(Model)
    .Name("chart")
    .Series(series => {
        series.Bar(s => s.RepSales).Name("Representative Sales");
        series.Bar(s => s.TotalSales).Name("Total Sales");
    })
    .ClientEvents(events => events
       .OnLoad(() => 
       {
        %>
         function(e) {
             // Perform required actions here.
         }
        <%
       }
    )
%>
  3. Declare the event handler inline (Razor syntax): 
    @(Html.EasyUI().Chart(Model)
    .Name("chart")
    .Series(series => {
        series.Bar(s => s.RepSales).Name("Representative Sales");
        series.Bar(s => s.TotalSales).Name("Total Sales");
    })
    .ClientEvents(events => events
        .OnLoad(
        @<text>
            function(e) {
                // Perform required actions here.
            }
        </text>)
    )
)

The chart has the following events:

  • OnLoad - raised when the component is initialized
  • OnDataBound - raised when the component has received data via Ajax binding and is about to render it.
  • OnSeriesClick - raised when the user clicks a data series point
