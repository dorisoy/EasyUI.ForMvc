This example shows "page on scroll" feature of EasyUI Grid for ASP.NET MVC. When the grid is scrolled to the bottom next page is requested. The response is accumulated and the gird is re-bound.

To enable this feature you need to use PageOnScroll method: 

<%= Html.EasyUI().Grid(Model)
        .Name("Grid")
        .Pageable(paging => paging.Style(GridPagerStyles.Status).PageOnScroll(true))
 %>
