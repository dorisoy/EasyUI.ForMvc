EasyUI Grid for ASP.NET MVC Key Features

  • Light HTML output
  • Minimal JavaScript footprint
  • Seventeen built-in themes
  • Flexible databinding - server, ajax and web service
  • Linq based expression engine for pushing grid operations (paging, sorting, filtering and grouping) to the database server
  • Paging with different pager styles
  • Single and multiple column sorting
  • Filtering based on column data type
  • Grouping
  • In-line data editing (insert/update/delete)
