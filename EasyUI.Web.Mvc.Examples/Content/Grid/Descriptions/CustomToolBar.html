This example demonstrates how to use a custom toolbar template in EasyUI Grid for ASP.NET MVC. The template can be declared through the ToolBar's Template method. In this particular example the template is used as a container for a filtering ComboBox.

To define a custom template use the following code 

<% Html.EasyUI().Grid(Model)
        .Name("Grid")
        .ToolBar(toolBar => toolBar.Template(() =>
        { 
            %>
            <label>Show orders for customer:</label>
            <%= Html.EasyUI().ComboBox()
                    .Name("Customers")                                               
            %>
            <% 
        }))
        .Render();
%>
