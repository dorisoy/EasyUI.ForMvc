This example shows how to use EasyUI Grid for ASP.NET MVC foreign key column to show related data.

The related data should be provided as either SelectList or IEnumerable: 

  Html.EasyUI().Grid>ClientEditableOrder>()
           .Name("Grid")                      
           .Columns(columns =>
           {
               columns.Bound(o => o.OrderID).Width(100);
               columns.ForeignKey(o => o.EmployeeID, (IEnumerable)ViewData["employees"],
                   "ID", "Name").Width(230);
               columns.Bound(o => o.OrderDate).Width(150);
               columns.Bound(o => o.Freight).Width(220);               
           })

When switched to edit mode, by default EasyUI DropDownList for ASP.NET MVC will be shown as column editor. This can be changed by modifing the GridForeignKey editor template.
