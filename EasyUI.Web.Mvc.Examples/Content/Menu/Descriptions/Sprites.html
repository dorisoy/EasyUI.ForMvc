This example shows how to configure EasyUI Menu for ASP.NET MVC to use sprites.

Use SpriteCssClasses method to set the css class name. The menu will render a <span> tag with the provided css classes before the item's text: 

<%= Html.EasyUI().Menu()
        .Name("Menu")
        .Items(items => 
        {
            items.Add()
                 .Text("EasyUI Extensions for ASP.NET MVC")
                 .SpriteCssClasses("icon-mvc")
                 .Items(parent =>
                 {
                     parent.Add().Text("Grid")
                           .SpriteCssClasses("icon-mvc-grid");
                 ...
                 }
        ...
        });
%>
