This example shows the accessibility features of EasyUI ComboBox for ASP.NET MVC. The following code snippet shows how to do this: 

<%= Html.EasyUI().ComboBox()
        .Name("ComboBox")
        ...
%>

Note:

EasyUI ComboBox for ASP.NET MVC is level A accessible (in compliance with the W3C Web Accessibility Guidelines 2.0).

The DropDownList is not currently accessible in scenarios with disabled JavaScript.
