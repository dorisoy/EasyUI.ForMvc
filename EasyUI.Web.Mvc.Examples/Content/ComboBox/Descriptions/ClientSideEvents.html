This example shows the client-side events supported by EasyUI ComboBox for ASP.NET MVC.

The combobox has the following events:

  • OnLoad - raised when the component is initialized
  • OnOpen (cancellable) - raised when a suggestion list is opened
  • OnClose (cancellable) - raised when a suggestion list is closed
  • OnChange - raised when the selected value is changed.
  • OnDataBinding (cancellable) - raised a request for data has been made (load on demand)
  • OnDataBound - raised a request for data has finished
  • OnError - raised if an error occurs while request (load on demand) for data is made.

AutoComplete and DropDownList support same list of client events.

You can subscribe to each of the events in the following ways:

  • With a pre-defined javascript function 
    <%= Html.EasyUI().ComboBox()
        .Name("ComboBox")
        .ClientEvents(events => events
            .OnChange("onChange")
        )
%>

<script type="text/javascript">
    function onChange(e) {
        // your code
    }
</script>
  • With an inline function, serialized from the server 
    <% Html.EasyUI().ComboBox()
       .Name("ComboBox")
       .ClientEvents(events => events
       .onChange(() => {%> function(e) { // your code } <%})
       .Render();
%>
  • Purely client-side event registration 
    <%= Html.EasyUI().ComboBox().Name("ComboBox") %>

<script type="text/javascript">
    $(document).ready(function() {
        $('#ComboBox').bind('valueChange', function(e) { // your code });
    });
</script>
