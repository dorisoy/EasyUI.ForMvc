This example shows how to configure EasyUI ComboBox for ASP.NET MVC to load its data through Ajax requests. Server should return JSON data.

You can bind the combobox to an Action method that will get requested through AJAX, or to an ASMX / WCF web service, using the DataBinding method. 

<%= Html.EasyUI().ComboBox()
     .Name("ComboBox")
     .DataBinding(binding => binding.Ajax().Select("_AjaxLoading", "ComboBox"))
 %>

You can load whole list of items using the BindTo method or generate it through Items method.

Available settings:
  • Cache:
    • Enabled: Last recieved data will be saved and any additional filtration will be done perform on it.
      If there are no matches after filtering, combobox will initiate another ajax request.
    • Disabled: ComboBox will perform ajax request on every keystroke. 
            <%= Html.EasyUI().ComboBox()
                .Name("ComboBox")
                .DataBinding(binding => binding.Ajax().Select("_AjaxLoading", "ComboBox").Cache(true))
            %>

    AutoComplete component behaves similarly.
    DropDownList will make ajax request only once and cache the recieved data.

  • Delay: The delay in milliseconds which combobox will wait after a keystroke to initiate ajax request.
