EasyUI ComboBox for ASP.NET MVC Key Features

  • Light HTML output
  • Seventeen built-in themes
  • Keyboard navigation
  • Filtering based on "StartsWith" and "Contains" filtering options
  • Flexible databinding - server, ajax and web service
  • Client API and event handling
  • Animation effects
  • There are also DropDownList and AutoComplete components except ComboBox.
    • AutoComplete component has multiple values support.
    • DropDownList component is a lightweight version of ComboBox.
