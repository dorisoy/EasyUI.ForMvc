This example shows how to nest other components in EasyUI Splitter for ASP.NET MVC.

To nest other components in the splitter, use the Content method: 

<% Html.EasyUI().Splitter()
        .Name("Splitter")
        .Content(() =>
        {%>
            <%= Html.EasyUI().PanelBar()
                .Name("PanelBar")
                /* ... */
            %>
        <%});
%>

Nesting of splitters is supported. This way, you can have a mixed vertical / horizontal layout.
