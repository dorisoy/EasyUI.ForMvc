This example shows how to configure the EasyUI Splitter for ASP.NET MVC to render in horizontal or vertical direction.

Use the Orientation method to set the orientation of the splitter. The method accepts the SplitterOrientation enumeration:

  • Horizontal orientation : Use the SplitterOrientation.Horizontal option 
    <%= Html.EasyUI().Splitter()
        .Name("Splitter")
        .Orientation(SplitterOrientation.Horizontal)
        ...
%>
  • Vertical orientation : Use the SplitterOrientation.Vertical option 
    <%= Html.EasyUI().Splitter()
        .Name("Splitter")
        .Orientation(SplitterOrientation.Vertical)
        ...
%>
