This example shows the client-side events supported by EasyUI Splitter for ASP.NET MVC.

The splitter has the following events:

  • OnLoad - raised when the component is initialized
  • OnResize - raised when a pane is resized
  • OnExpand (cancellable) - raised when a pane gets expanded
  • OnCollapse (cancellable) - raised when a pane gets collapsed
  • OnContentLoad - raised when a request for the pane contents has finished

You can subscribe to each of the events in the following ways:

  • With a pre-defined javascript function 
    <%= Html.EasyUI().Splitter()
        .Name("Splitter")
        .ClientEvents(events => events
            .OnResize("onResize")
        )
%>

<script type="text/javascript">
    function onResize(e) {
        // your code
    }
</script>
  • With an inline function, serialized from the server 
    <% Html.EasyUI().Splitter()
       .Name("Splitter")
       .ClientEvents(events => events
       .OnLoad(() => {%> function(e) { /* your code */ } <%})
       .Render();
%>
  • Purely client-side event registration 
    <%= Html.EasyUI().Splitter().Name("Splitter") %>

<script type="text/javascript">
    $(document).ready(function() {
        $('#Splitter').bind('contentLoad', function(e) { /* your code */ });
    });
</script>
