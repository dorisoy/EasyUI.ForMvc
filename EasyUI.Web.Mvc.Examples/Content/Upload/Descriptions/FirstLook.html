EasyUI Upload for ASP.NET MVC Key Features

  • Asynchronous and synchronous (on form submit) file upload
  • Multiple file selection
  • Progress tracking *
  • Drag and drop *
  • Cancelling upload in progress *
  • Removing uploaded files
  • Seventeen built-in themes
  • Client API and event handling
  • Localization
  • Standards based, no plug-ins required

* These features will be automatically turned on if available in the browser. Consult the documentation for a list of supported browsers.
