This example shows the client-side events supported by EasyUI Upload for ASP.NET MVC.

You can attach an event handler using either of these options:

  1. Specifiy the JavaScript event handler name: 
    Html.EasyUI().Upload()
    .Name("attachments")
    .ClientEvents(events => events
        .OnLoad("onLoad")
    )
  2. Declare the event handler inline (WebForms syntax): 
    <%= Html.EasyUI().Upload()
    .Name("attachments")
    .ClientEvents(events => events
       .OnLoad(() => 
       {
        %>
         function(e) {
             // Perform required actions here.
         }
        <%
       }
    )
%>
  3. Declare the event handler inline (Razor syntax): 
    @(Html.EasyUI().Upload()
    .Name("attachments")
    .ClientEvents(events => events
        .OnLoad(
        @<text>
            function(e) {
                // Perform required actions here.
            }
        </text>)
    )
)

The upload has the following events:

  • OnLoad - raised when the component is initialized
  • OnSelect - raised when one or more files are selected (cancelable)
  • OnUpload - raised when one or more files are about to be uploaded (cancellable)
  • OnSuccess - raised when an upload / remove operation has been completed successfully
  • OnError - raised when an upload / remove operation has failed
  • OnComplete - raised when all active uploads have completed either successfully or with errors
  • OnCancel - raised when the upload has been cancelled while in progress
  • OnRemove - raised when an uploaded file is about to be removed (cancellable)
