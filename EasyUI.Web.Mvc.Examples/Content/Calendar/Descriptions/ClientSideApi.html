This example shows how to use the client-side API of EasyUI Calendar for ASP.NET MVC.

To use the client-side API you should first retrieve the calendar client-side object. The following code snippet shows how to do this: 

<script type="text/javascript">
function getCalendar(){
    //"Calendar" is the value specified by the Name() method.
    var calendar = $("#Calendar").data("tCalendar");
    return calendar;
}
</script>

Available client-side methods:

  • You can set selected date with the value(...) method. This method accepts Date object or string.
  • You can retrieve selected date with value() method. This method returns Date object or null.
