This example shows the templates support in EasyUI TreeView for ASP.NET MVC.

You can set the template for the contents of a given item in the following way:

<% Html.EasyUI().TreeView()
        .Name("TreeView")
        .Items(treeview =>
        {
            treeview.Add().Text("UI Components")
                .Content(() =>
                {%>
                    <!-- template content -->
                <%});
        })
        .Render();
%>
