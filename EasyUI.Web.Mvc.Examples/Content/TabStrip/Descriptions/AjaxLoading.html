This example shows how to configure EasyUI TabStrip for ASP.NET MVC to load the templated contents with Ajax requests. The request is performed only once. After that the content is rendered on the page and TabStrip shows/hides it without performing additional Ajax requests.

To set the item content you should use the LoadContentFrom method. 

<%= Html.EasyUI().TabStrip()
     .Name("TabStrip")
     .Items(items => 
     {
            items.Add()
                 .Text("Completely Open Source")
                 .LoadContentFrom("AjaxView_OpenSource", "TabStrip");
     })
 %>
